## 本地开发

### 环境要求

- Node.js 18.0 或更高版本
- pnpm 8.0 或更高版本

### 安装依赖

```bash
# 安装项目依赖
pnpm install
```

### 启动开发服务器

```bash
# 启动本地开发服务器
pnpm docs:dev
```

### 构建文档

```bash
# 构建静态文件
pnpm docs:build

# 预览构建结果
pnpm docs:preview
```
